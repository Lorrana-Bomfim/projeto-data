# 📅 Conversor de Datas (Americano ⇄ Brasileiro)

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

Aplicação web simples desenvolvida para converter formatos de datas de forma prática, utilizando JavaScript para a lógica de manipulação e HTML/CSS para a interface.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação da página e campos de entrada de dados.
- **CSS3:** Estilização básica e layout responsivo.
- **JavaScript:** Lógica de manipulação de strings/datas e interação com a tela (DOM).
